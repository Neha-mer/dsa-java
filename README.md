# DSA & Java Concepts in Java # Inprogress

A hands-on repository for practicing **Data Structures & Algorithms (DSA)**, **core Java concepts**, **multithreading**, and **coding interview problems** — all implemented in Java.

## 🧩 Topics Covered

### 🔢 DSA Patterns
- **Two Pointer** — Pair sum, sorted array problems
- **Sliding Window** — Fixed and variable size windows
- **Prefix Sum** — Range queries and cumulative sums
- **Arrays** — Searching, sorting, and manipulation

### 💡 Coding Interview Problems
| Problem | Technique |
|---|---|
| Two Sum | HashMap / Two Pointer |
| Three Sum | Sorting + Two Pointer |
| Longest Palindrome Substring | Expand Around Center |
| Longest Substring Without Repeating | Sliding Window |
| Merge Two Sorted Arrays | Two Pointer |
| Find Median of Sorted Array | Binary Search |
| Find Missing Number | Math / XOR |
| Anagram Check | Sorting / Frequency Map |
| Sum to Target | Two Pointer |

### ☕ Core Java
- OOP: Inheritance, Interfaces, Constructors, Constructor Chaining
- Immutable Classes
- Singleton Pattern
- `Comparable` vs `Comparator`
- String manipulation & Array utilities

### 🌊 Java 8
- Stream API: `map()`, `flatMap()`

### 🔒 Concurrency & Multithreading
- Thread creation
- Race conditions
- Deadlocks
- `ReentrantLock`
- Producer-Consumer problem
- Virtual Threads (Java 21+)

### ⚠️ Exception Handling
- Custom exception handling patterns

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd dsa-java
   ```

2. **Open in IntelliJ IDEA** (or any Java IDE)

3. **Run any class** that contains a `main()` method directly from your IDE.
